📖 概要 / Overview

📘 日本語

本グラフは、ローグライクカードゲーム 『東方光耀夜 (LBoL)』 における 各カードのパフォーマンスや共起関係を可視化したものです。 使用率・同時採用傾向・ルート選択など分析しています。

📗 English

This graph visualizes the performance and co-occurrence relationships of each card in the roguelike deck-building game “Touhou Lost Branch of Legend (LBoL)”. It analyzes factors such as usage rate, simultaneous adoption tendencies, route selection, and so on.

🧮 集計方法 / Methodology

📘 日本語

1. ログの収集とフィルタリング

公開されているログリポジトリ(lbol logs)から最新のログデータを自動で収集します。 その際、以下の条件で分析に適したログだけを厳選しています。

  • Lunatic7 真エンドクリアのログのみを使用
  • 一部MODを除外（ゲームバランスに影響を与えるMODが使用されたログは除外）
  • 不正が疑われるログの除外（異常なステータス変動・不連続なデータなど）

2. プレイヤーごとのログ抽出（サンプリング）

特定のプレイヤーのログが多すぎると、その人のプレイスタイルが全体に大きく影響してしまいます。 これを防ぐため、各プレイヤーから抽出するログ数に上限を設けています。

ログ数が上限に満たないプレイヤーについては制限を適用せず、すべての有効ログを使用します。 上限を超えるプレイヤーに対しては、分位加重サンプリングによってログを選び出します。

  1. ログを新旧4グループに分割（最古・やや古い・やや新しい・最新の25%ずつ）
  2. 新しいグループから多めに、古いグループから少なめにランダム抽出

3. カード性能の計算

各カードが戦闘に与える影響を「状況別」に評価します。

  • 状況の定義: Actごとの進行度や敵種別（例：Act1の雑魚戦、Act2のエリート戦）で分類
  • 性能の算出: カードを所持していた場合と所持していなかった場合の平均クリアターン・HPロスを比較し、貢献度を計算
  • 信頼性の担保:
    • 戦闘回数が一定数以上（例：雑魚戦20回、ボス戦10回）のカードのみ評価
    • ベイズ平均により、サンプル数の少ないカードの極端な値を補正

4. 総合評価とランキングの作成

状況別に計算された性能スコアを以下の指標で総合評価し、ダッシュボードに表示します。

  • 総合パフォーマンス（バブルグラフ）: 各状況の性能を「動的重み付け」で加重平均。採用率や戦闘回数の基準を満たすカードを表示。 低採用カードでも評価に参加できるよう、ソフトフィルターを導入し、ランキングの安定性を向上。
  • Act別パフォーマンス: 各Actで特に強力なカードをランキング形式で表示。
  • 傾向値（上振れ度）: 特定条件下におけるカードの上振れ下振れを示す指標。
    • プラス: 0.5を超えるとかなり強い上昇傾向にあります。
    • 攻撃傾向: 攻撃バフやバリアのような挙動をするものはこれに分類されやすい。
    • 防御傾向: 防御バフ、永き夜対策、廃棄、マナカードはこれに分類されやすい。
  • 注目度スコア: 「採用率」「性能」「ハイライト回数」を総合評価し、注目すべきカードを選出。
📗 English

1. Log Collection and Filtering

The system automatically collects the latest public log data from the repository(lbol logs). Only logs suitable for analysis are selected under the following conditions:

  • Only logs from Lunatic7 True Ending clears are used.
  • Logs involving balance-altering mods are excluded.
  • Logs suspected of cheating (abnormal stats, data discontinuity, etc.) are automatically filtered out.

2. Player Log Sampling

If one player contributes too many logs, their playstyle could dominate the results. To prevent this, a limit is imposed on the number of logs sampled per player.

Players below the limit are unaffected — all their valid logs are used. For players exceeding the limit, logs are selected using quantile-weighted sampling:

  1. Divide the player’s logs into four chronological groups: oldest 25%, slightly old 25%, slightly new 25%, newest 25%.
  2. Sample more logs from newer groups and fewer from older ones, randomly within each group.

3. Card Performance Evaluation

Each card’s contribution to battle outcomes is evaluated by context.

  • Context Definition: Categorized by game progress and enemy type (e.g., Act 1 normal battles, Act 2 elite battles).
  • Performance Calculation: Compares average clear turns and HP loss with and without the card, measuring its contribution.
  • Reliability Measures:
    • Only cards used in a sufficient number of battles (e.g., 20 normal, 10 boss) are evaluated.
    • Bayesian averaging corrects extreme values caused by low sample counts.

4. Overall Rating and Ranking

Context-based performance scores are combined into comprehensive indicators displayed on the dashboard:

  • Overall Performance (Bubble Chart): Weighted average using dynamic weighting across all contexts. A soft filter smooths influence from low-adoption cards, stabilizing the rankings.
  • Performance by Act: Highlights cards that perform exceptionally well in specific Acts.
  • Variance Index (Swing Potential): Shows how consistent or volatile a card’s performance is:
    • Positive: When it exceeds 0.5, it indicates a fairly strong upward trend.
    • Offensive tendency: Things that behave like attack buffs or barriers are likely to be classified into this category.
    • Defensive tendency: Defense buffs, Eternal Night countermeasures, exhaust effects, and mana cards are likely to be classified into this category.
  • Attention Score: A composite indicator combining adoption rate, performance, and highlight frequency to identify noteworthy cards.
