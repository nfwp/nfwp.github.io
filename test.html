The data is aggregated from public logs. To prevent bias, the number of runs from the single most active player for each character is adjusted so that it accounts for less than a fixed percentage of the total runs. (If a player has too many plays and some runs need to be truncated, the newer ones are prioritized.) Data that appears inconsistent (suspected cheating) or that comes from certain mods or farming behaviors is excluded. The evaluation of each card is slightly adjusted to be closer to human intuition, so it is not absolute; however, the overall relationships should be roughly accurate. The overall rating also reflects dynamic weighting based on the context of each play.
